# Auth Api

Created by Sarah Creager

## Installation
`npm install`  

*sample .env file:*
``` 
PORT=3001

SECRETL=
```

## Dependencies
 * eslint
 * express
 * jest
 * method-override
 * pg
 * sequelize
 * sequelize-cli
 * sqlite3
 * base-64
 * bcrypt
 * cors
 * dotenv
 * jsonwebtoken
 * morgan
 * supertest

## Summary of Problem Domain
Built basic auth server that allows RESTful CRUD, authentication with username and password and authorization with JWT.
